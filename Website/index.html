Diferentele dintre Kernel si Shell

Pe scurt diferența dintre Kernel și Shell este că acest Kernel este nucleul sistemului de operare care controlează toate sarcinile sistemului, în timp ce Shell-ul este interfața care permite utilizatorilor să comunice cu nucleul. Pe scurt, Shell-ul este interfața care ajută utilizatorii să acceseze serviciile furnizate de sistemul de operare.
