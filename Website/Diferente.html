Utilizare

Un Kernel este un program de nivel scăzut care se interfață cu hardware-ul (CPU, RAM, discuri, rețea, ...) pe care se execută aplicații. Este cel mai scăzut program care rulează pe computere, deși cu virtualizare puteți avea mai multe nuclee care rulează pe mașini virtuale care rulează ele însele pe un alt sistem de operare.

Un Shell (CLI) este un interpret de comandă, adică programul care fie procesează comanda pe care o introduceți în linia de comandă (aka terminal), fie procesează scripturi shell (fișiere text care conțin comenzi) (modul batch). La începutul timpului Unix, era un mod unic de a interacționa cu utilizatorii. În zilele noastre, interfețele grafice de utilizator (GUI) devin tipul de shell preferat pentru majoritatea utilizatorilor.

Tipuri

Kernel nu are diferite tipuri. Shell are diferite tipuri, cum ar fi Bourne shell, C shell, Korn Shell, Bourne Again Shell etc.
